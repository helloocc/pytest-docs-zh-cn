pytest: 助你更好编程¶

pytest 框架可以轻松编写小型测试，同时支持应用程序和软件库的复杂功能性测试。

一个简单的例子：

# content of test_sample.py
def inc(x):
    return x + 1


def test_answer():
    assert inc(3) == 5

执行：

$ pytest

collected 1 item

================================= FAILURES =================================

    def test_answer():
>       assert inc(3) == 5
E       assert 4 == 5
E        +  where 4 = inc(3)

test_sample.py:6: AssertionError

由于 pytest 有详细的断言说明，所以只简单使用 assert 语句。有关更多示例，请参阅 入门

特性¶

  • 失败的 断言语句 详细信息（无需记住 self.assert* 名称）;

  • 自动发现 测试模块和功能;

  • 模块化fixtures 用于管理小型或参数化长生命周期的测试资源;

  • 可以运行 unittest （包括试用期）,并且可以开箱即用测试套件 nose;

  • Python 3.5+ and PyPy 3;

  • 丰富的插件架构，拥有超过315+ 外部插件 的蓬勃发展的社区;
